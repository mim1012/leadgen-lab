# 프로필 이미지 프롬프트: 집 아이콘 + 체크 표시

목표: 네이버 블로그, 네이버 카페, Threads에서 광고계정처럼 보이지 않고 “이사 견적 전 체크리스트를 정리해주는 정보 계정”으로 보이게 한다.

## 최종 추천 프롬프트

```text
Create a clean, trustworthy profile icon for a Korean moving checklist account. A simple rounded house icon combined with a clear check mark, suggesting “check before requesting a moving quote.” Minimal vector logo style, friendly but professional, soft navy and warm green color palette, white or very light background, centered composition, high contrast, no clutter, no realistic people, no truck, no company branding, no price claims. The icon must remain readable as a small circular social media avatar. Add very short Korean text only if legible: “견적 전 체크”. Square 1:1 format.
```

## 한글 프롬프트

```text
한국어 이사 체크리스트 계정용 프로필 이미지를 만들어줘. 둥근 집 아이콘과 선명한 체크 표시를 결합해서 “이사 견적 신청 전에 확인한다”는 느낌이 나야 해. 미니멀한 벡터 로고 스타일, 친근하지만 전문적인 분위기, 부드러운 네이비와 따뜻한 그린 색상, 흰색 또는 아주 밝은 배경, 중앙 정렬, 작은 원형 프로필 썸네일에서도 잘 보여야 함. 사람 얼굴, 이삿짐 트럭, 과한 광고 문구, 최저가/무료/1등 같은 문구는 제외. 텍스트가 들어간다면 아주 짧게 “견적 전 체크”만 사용. 1:1 정사각형.
```

## 텍스트 없는 버전

```text
Clean minimalist vector profile icon, rounded house symbol with a bold check mark, trustworthy moving checklist account, soft navy and warm green, white background, centered, high contrast, readable at small circular avatar size, no text, no people, no truck, no price or promotional claims, square 1:1.
```

## 피해야 할 요소

- “최저가”, “무료견적”, “전국 1등” 문구
- 실제 업체 로고처럼 보이는 과한 상업 디자인
- 트럭/상담원/전화기 중심 이미지
- 작은 썸네일에서 안 보이는 긴 문장
- 너무 AI 캐릭터 느낌 나는 3D 인물

## 적용 추천

- 네이버 블로그/카페/Threads 공통 사용: 텍스트 없는 버전 우선
- 네이버 블로그 대표 이미지나 고정글 썸네일: “견적 전 체크” 텍스트 버전 사용
- 배경색: 흰색 또는 연한 아이보리
- 포인트 색: 체크 표시만 그린 계열
